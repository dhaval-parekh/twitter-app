Demo Twitter Application Developed in PHP.

### Requirement

* 	Php Extension
	-	`JSON`
	-	`sqlite3`
	-	`mcrypt`
	
### Installation

* 	update config/config.php Put your domain name in `BASE_URL`, and twitter application's cradential.
* 	you have to give full access to following directory
	- 	`tmp/session/`
	- 	`uploads/`
	-	`db/`
